# Nerds
Final project of 'Professional web interface development' course by [HTML Academy](https://htmlacademy.ru/).

## Project features
* Semantic HTML
* Flex
* Based on PSD template
* Image optimization
* Pixel Perfect
* Accessibility
* Some JS (mouse-events, key-events)
